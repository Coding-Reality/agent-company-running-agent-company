# agent-company-running-agent-company

A Redmine-native agent company whose sole product is [base-agent-company](https://github.com/Coding-Reality/base-agent-company).

## What This Is

This is a **real operating company** for the product. The company's job is to:

- **Market** the agent-company concept to developers, teams, and enterprises
- **Expand** the idea with use cases, architecture guides, and deployment patterns
- **Write** blogs, tutorials, and thought-leadership content about autonomous agent companies
- **Manage** the public `base-agent-company` repo — issues, roadmap, releases, community
- **Build** awareness and adoption of the framework

It is intentionally meta, but this repo is no longer the primary execution layer for the company.

## Product

**[base-agent-company](https://github.com/Coding-Reality/base-agent-company)** — a ready-to-fork template for creating autonomous companies as folders. This project markets that product, but the company itself now runs through Redmine rather than a filesystem-based agent workflow.

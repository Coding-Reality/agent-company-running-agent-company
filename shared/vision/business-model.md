# Business Model

## Product
**base-agent-company** — open-source framework for filesystem-based autonomous companies.

GitHub: https://github.com/Coding-Reality/base-agent-company

## Revenue Streams (Near-Term)
1. **Consulting** — help teams deploy and customize agent-company instances for their workflows
2. **Premium templates** — industry-specific or advanced role configurations sold as add-ons
3. **Enterprise support** — SLA-backed setup, maintenance, and optimization for companies running production agent-company deployments
4. **Sponsored content** — partnering with AI tool vendors to create integration guides and tutorials

## Revenue Streams (Medium-Term)
5. **Training & workshops** — paid sessions teaching teams to build and operate agent companies
6. **Managed hosting** — run-your-agent-company-as-a-service offering

## Growth Model
Adoption-first: grow the open-source user base through exceptional content, then convert a fraction into paying consulting/enterprise customers.

## Key Metrics
- GitHub stars, forks, and clones on base-agent-company
- Blog/tutorial traffic and engagement
- Inbound consulting inquiries
- Community size (Discord, GitHub Discussions, etc.)
